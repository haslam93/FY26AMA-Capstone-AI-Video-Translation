# Capstone Project

## Overview
This project provides video translation services using Azure Durable Functions.

## Project Structure
- `/src` - Source code
  - `/Api` - HTTP endpoints for submit and status
  - `/Orchestrator` - Durable Functions orchestrations and state machine
  - `/Workers` - Worker functions for various tasks
  - `/Shared` - Shared contracts, telemetry, and error models
- `/tests` - Unit and integration tests
- `/infra` - Infrastructure as Code (Bicep)
- `/docs` - Documentation
- `/.github/workflows` - CI/CD pipelines
